This is the most common type of task in our courses.

Its design is straightforward: there are task files that need to be modified and test files that validate the written code.

### Tests

`tests/test.py` — Contains tests that will be executed when the `Check` button is clicked.

### Placeholders
Since `main.py` contains the correct solution to the task, all that remains is to turn this into a challenge for the learner. The plugin allows you to hide certain parts of the code and display placeholders, indicating where the learner should write their solution. 

### Read mode

Check out the [Edu task](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/educator-start-guide.html#edu_task) section in the Course creator start guide for more details about this task type and placeholders.
