With this task type, you can create a quiz that allows learners to select either a single answer or multiple answers. Use it as a quick knowledge check to ensure learners are keeping up with the material.

A Quiz task should be written in the `task.md` file, while the possible answers are defined in the `task-info.yaml` file. Check out the [Multiple-choice task](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/educator-start-guide.html#multiple_choice_task) section in the Course creator start guide for more details.

### Task example
Which programming language template are you looking at?
